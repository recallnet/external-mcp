# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2024-11-17

### Initial Release

- Modular MCP server implementation with separate modules for Twitter, Substack, and CoinGecko
- Comprehensive TypeScript interfaces and documentation
- Support for both individual module usage and combined server
- Integration with Claude via MCP
- HTML processing utilities for Substack content

### Features

- Twitter module with support for reading tweets, profiles, and trends
- Substack module with support for publications, posts, and comments
- CoinGecko module with support for token prices, contracts, and trends
- Shared utilities for HTML processing and common operations
- Combined server with conditional module loading
